# モデル

```text
data 依頼 = プロジェクト依頼 
  OR タスク依頼
  OR コンペ依頼

data 依頼ベース = 依頼タイトル
  AND 依頼詳細
  AND 求めるスキル?
  AND 添付ファイル?
  AND List<特記事項>
  AND 応募期限

data 支払い方式 = 固定報酬制 OR 時間単価制
data 予算金額範囲 = 5千円未満
  OR 5千円〜1万円
  OR 1万円〜3万円
  OR 3万円〜5万円
  OR 5万円〜10万円
  OR 10万円〜15万円

data 予算 = 予算金額範囲
  OR 予算指値
  OR 指定しない

data 時間単価 = 700円〜1000円
  OR 1000円〜1500円
  OR 1500円〜2000円
  OR 2000円〜3000円
  OR 3000円〜4000円
  OR 4000円〜5000円
  OR 5000円以上
  
data 週の稼働時間 = Integer // 1〜160
data 依頼期間 = 1週間以内
  OR 1週間〜1ヶ月
  OR 1ヶ月〜3ヶ月
  OR 3ヶ月〜6ヶ月
  OR 6ヶ月以上

data 週の稼働と依頼期間 = 週の稼働時間 AND 依頼期間

data 募集人数 = Integer // 1〜1024

data 固定報酬制 = 募集人数
  AND 予算

data 時間単価制 = 募集人数
  AND 時間単価
  AND 週の稼働と依頼期間

data 応募期限 = 日付 // 依頼日時から14日後までの範囲の日付
data 納品希望日 = 日付 // 依頼日時以降の日付
data プロジェクト依頼 = 依頼ベース
  AND 支払い方式
  AND 応募期限
  AND 納品希望日

data 作業単価 = Integer // 5〜
data 作業件数 = Integer // 1〜
data 作業単価・件数 = 作業単価
  AND 作業件数

data 1人あたりの件数上限 = Integer // 0以上
data 制限をかける = 1人あたりの件数上限
data 1人あたりの作業件数 = 制限をかけない
  OR 制限をかける

data タスク依頼 = 依頼ベース
  AND 作業単価・件数
  AND 1人あたりの作業件数

data カスタム契約金額 = 契約金額指値

data 契約金額 = エコノミー
  OR ベーシック
  OR スタンダード
  OR プレミアム
  OR カスタム契約金額

data コンペ依頼 = 依頼ベース
  AND 契約金額
```
